# 客服自动化技能

自动化处理客服常见场景：订单查询 → 状态分析 → 生成回复 → 记录日志

## 工作流程

### 步骤 1：订单状态查询
- **使用工具**: `mysql_query`
- **参数**:
  - `sql`: "SELECT order_id, status, create_time, total_amount FROM orders WHERE order_id = '{orderId}'"
  - `limit`: 10

### 步骤 2：物流信息查询（可选）
- **使用工具**: `logistics_query`
- **参数**:
  - `orderId`: "{orderId}"
- **触发条件**: 用户询问物流、快递、发货时间

### 步骤 3：生成回复
- 解析订单数据
- 根据状态生成友好回复

### 步骤 4：记录工单
- 如遇问题，自动创建工单记录

## 触发关键词

- 订单查询类：我的订单、查一下订单、订单到哪了
- 物流类：物流信息、快递到哪了、什么时候到
- 售后类：退货、退款、换货、售后问题
- 催单类：快点发货、还没到、催一下

## 注意事项

- 所有查询需先验证用户身份（手机号/订单号）
- 如订单不存在，引导用户提供正确信息
- 情绪化用户先安抚，再解决问题
- 复杂问题转人工处理
