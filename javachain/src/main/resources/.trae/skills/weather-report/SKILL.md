# 天气查询格式化技能

此技能提供完整的天气查询工作流程，并输出格式化结果。

## 工作流程步骤

### 步骤 1：查询当前天气（当用户询问当前天气时）
- **使用工具**: `weather_query`
- **参数**:
  - `city`: 城市名称（如 "北京"）

### 步骤 2：查询天气预报（当用户询问预报）
- **使用工具**: `weather_forecast`
- **参数**:
   - `city`: 城市名称
   - `days`: 3

## 使用示例

**用户**: "北京今天天气怎么样？"
**技能响应**: 使用 `weather_query` 查询城市天气

**用户**: "上海未来3天天气预报"
**技能响应**: 使用 `weather_forecast` 查询天气预报

## 注意事项

- 如果用户未指定城市，请先询问
- 如果 API 返回错误，提供友好的错误提示
